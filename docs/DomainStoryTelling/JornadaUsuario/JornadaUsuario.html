FIAP Cloud Games - Jornada do Usuário

Tech Challenge - FIAP Cloud Games (FCG) Jornada do Usuário Cadastro e uso da plataforma por um usuário comum 1. O Usuário acessa a plataforma. 2. O Usuário preenche seu nome, e-mail e senha para se cadastrar. 3. O sistema valida o e-mail e a senha segura. 4. O sistema salva o novo usuário no banco de dados. 5. O Usuário faz login com e-mail e senha. 6. O sistema valida as credenciais e gera um token JWT. 7. O Usuário acessa sua biblioteca de jogos adquiridos. 8. O Usuário pode visualizar os detalhes dos jogos da sua biblioteca.
