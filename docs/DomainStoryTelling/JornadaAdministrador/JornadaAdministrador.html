FIAP Cloud Games - Jornada do Administrador

Tech Challenge - FIAP Cloud Games (FCG) Jornada do Administrador Ações administrativas 1. O Administrador faz login na plataforma. 2. O sistema valida o login e emite um token JWT com role de admin. 3. O Administrador acessa o painel administrativo. 4. O Administrador pode cadastrar novos jogos. 5. O Administrador pode criar promoções para jogos. 6. O Administrador pode gerenciar usuários (ex: atualizar ou remover).
